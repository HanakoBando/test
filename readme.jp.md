# MQTT for DNEとは
MQTT for DNEは、Doog社製品のロボットのうちMQTTでの通信が可能な機種をブラウザ上から操作したり、機体の状態を取得することができるアプリケーションの例を示したものです。『ブラウザ版アプリ』と呼称することもあります。[テストページ](http://sample-app.doog-cloud.com/)で動作イメージをご覧いただくことができます。

# 必要なもの
- Doog社ロボット
    - 協働運搬ロボット
        - サウザーEシリーズ
        - メカロン
    - 搭乗型ロボット
        - ガルー
        - モビリス
- Javascriptを利用可能なブラウザ
    - Chrome(PC/スマホ)
    - Safari(iOS)
